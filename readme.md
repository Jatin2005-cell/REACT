# ⚛️ React Learning Projects

A collection of React projects built while learning React.js and modern frontend development. This repository showcases concepts ranging from React fundamentals to routing, state management, Context API, and reusable components.

## 🚀 Projects

| Project                   | Description                                             |
| ------------------------- | ------------------------------------------------------- |
| **01 Vite React**         | React project setup using Vite                          |
| **02 Counter App**        | State management using `useState`                       |
| **03 Tailwind Props**     | Components, props, and Tailwind CSS                     |
| **04 Background Changer** | Dynamic background color changer                        |
| **05 Password Generator** | Password generator using React Hooks                    |
| **06 Custom React**       | Understanding React internals and custom implementation |
| **07 React Router**       | Client-side routing with React Router                   |
| **08 Todo Context API**   | Todo application using Context API and Local Storage    |

## ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/Jatin2005-cell/React.git
```

Go to any project:

```bash
cd 05passwordGenerator
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
